# ichgramm

**Social Media App**

Full stack Instagram-style social media application: posts, likes, comments, follows, real-time chat. Built with React, Vite, Redux, Node.js, Express.js, TypeScript, MongoDB (Mongoose), WebSockets and JWT authentication.

- **Backend (this repo):** [https://github.com/dimovdmitrii/ichgramm_backend](https://github.com/dimovdmitrii/ichgramm_backend)

- **Live API:** [https://ichgramm-backend.onrender.com](https://ichgramm-backend.onrender.com)
